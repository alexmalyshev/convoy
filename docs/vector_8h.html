Detailed Description

Header for a dynamic array data structure library.

A vector is a dynamic array. Inserts and pushes into a vector will always succeed provided there is enough memory on the system. Elements are stored as generic pointers (void *), however NULL cannot be stored. Getting a NULL back as an element from a vector means that position in the vector is empty.
