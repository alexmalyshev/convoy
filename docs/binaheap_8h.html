Detailed Description

Function prototypes and struct for a binary heap data structure.

Implementation of a minimum binary heap backed by a dynamic array. Elements are stored as generic pointers (void *) where an element value of NULL cannot be stored. Elements are compared using a generic compare function

Author:
Alexander Malyshev
