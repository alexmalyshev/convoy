Detailed Description

Function prototypes and struct for a circular buffer data structure.

Implementation of a fixed-size circular buffer. Elements are stored as generic pointers (void *) where NULL cannot be stored, and the circbuf structure allocates an array of void *s with one extra as it makes it easy to tell the difference between an empty circbuf and a full circbuf. However this extra slot will never be used to store an element.
