Detailed Description

Header for a circular buffer data structure library.

A circbuf is a fixed-size array. Elements are stored as generic pointers (void *), however NULL cannot be stored. The circbuf structure allocates an array of void *s with one extra as it makes it easy to tell the difference between an empty circbuf and a full circbuf. However this extra slot will never be used to store an element. A circbuf will not resize when trying to enqueue on an element when it is full, the enqueue will simply fail and return 1. Getting a NULL back as an element from a circbuf means that the circbuf is empty.
