Detailed Description

Function prototypes and structs for a hash table.

Implementation of a hash table that is backed by a dynamic array and uses separate chaining to resolve collisions. Elements are stored as generic pointers (void *) where an element value of NULL cannot be stored. Elements are compared using a user-specified compare function and hashed using a user-specified hash function.
