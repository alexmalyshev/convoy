Detailed Description

Header for a red black tree data structure library.

An rbtree is binary tree. Elements are stored as generic pointers (void *), however NULL cannot be stored. Elements are compared using a generic compare function. Inserting into an rbtree will always succeed provided there is enough memory on the system. Getting a NULL back as an element from an rbtree means that key has no mapped value in the tree.
