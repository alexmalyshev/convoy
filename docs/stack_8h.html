Detailed Description

Header for a stack data structure library.

A stack is a singly linked list. Elements are stored as generic pointers (void *), however NULL cannot be stored. Inserting into a stack will always succeed provided there is enough memory on the system. Getting a ULL back as an element from a stack means that the stack is empty.
