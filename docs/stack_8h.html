Detailed Description

Function prototypes and structs for a stack data structure.

Implementation of a LIFO stack that is backed by a singly linked list. The nodes in the linked list store data as generic pointers (void *) where an element value of NULL cannot be stored.

Author:
Alexander Malyshev
